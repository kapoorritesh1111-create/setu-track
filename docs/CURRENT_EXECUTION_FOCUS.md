# SETU TRACK — Current Execution Focus

This repository is the active baseline for the current stabilization and productization pass.

## What matters now

1. Dashboard must behave like a contractor operations command center.
2. Payroll reporting must be decision-ready for finance, not just data-visible.
3. Analytics must surface budget risk, spend concentration, and workflow blockers clearly.
4. Exports must preserve audit trust from receipt creation through paid-state confirmation.
5. Approvals must be fast to triage, especially for flagged or locked pay-period scenarios.

## Immediate product bar

SETU TRACK should not try to beat Deel, Remote, or Rippling on breadth first.
It should beat them on clarity, calm daily operations, and project-linked payroll intelligence for contractor-heavy teams.
